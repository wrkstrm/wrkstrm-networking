# WrkstrmNetworking

WrkstrmNetworking provides HTTP, JSON, and WebSocket primitives shared across Wrkstrm
clients. It layers transport and request modeling atop WrkstrmFoundation defaults so the
network stack stays consistent across services.

## Installation

```swift
.package(url: "https://github.com/wrkstrm/WrkstrmNetworking.git", from: "1.0.0"),
```

## Usage

Add the library to your target dependencies and import it where you build requests:

```swift
import WrkstrmNetworking
```
